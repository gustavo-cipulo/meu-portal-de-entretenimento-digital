# 🎬 Meu Portal de Entretenimento Digital

Projeto desenvolvido para a atividade da Agenda 13 – CSS, utilizando HTML e CSS para criar uma página web moderna e organizada sobre entretenimento digital.

---

## 🚀 Objetivo

Criar uma página web utilizando:

- ✅ HTML5
- ✅ CSS Inline
- ✅ CSS Interno
- ✅ CSS Externo
- ✅ Flexbox
- ✅ Classes e IDs
- ✅ CSS Reset

---

## 📂 Estrutura do Projeto

projeto/

├── portal.html

├── estilo.css

├── reset.css

└── imagens/

## 🎥 Seções da Página

### 🍿 Filmes e Séries
Área dedicada a conteúdos cinematográficos e séries populares.

### 🎧 Música e Podcasts
Espaço voltado para música digital e podcasts.

### 🎮 Games e Tecnologia
Seção focada em jogos eletrônicos e novidades tecnológicas.
